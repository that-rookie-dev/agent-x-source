# Changelog

All notable changes to Agent-X will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] — 2026-06-03

### Added

- Initial release of Agent-X VS Code extension
- Sidebar chat interface with streaming responses
- 168 built-in tools (file operations, terminal, Git, search, web, etc.)
- Multi-provider support: OpenAI, Anthropic, Google, Ollama, LM Studio, DeepSeek, Groq, Mistral, Together, xAI, Fireworks, Perplexity, Azure, Cohere
- Sub-agent delegation with live progress tracking
- Plan mode with step-by-step approval workflow
- RAG workspace indexing with semantic search
- Steer messages for mid-execution guidance
- Background task queue with progress monitoring
- Persistent agent memory (global + per-crew)
- Agent diary with highlights and insights
- SOUL.md personality anchor editor
- Session management with history, duplication, export, checkpoint/restore
- Permission system with risk-based approval
- Scope guard for workspace file access
- Scheduled reminders (one-shot and recurring)
- MCP server management with tool testing
- Skills and reflective learnings viewer
- Session mode switching (Agent / Ask / Plan)
- Crews and personas for different agent configurations
- Configuration sync between VS Code settings and config file
- Bidirectional file watcher for config changes
- Crash recovery with auto-save
- Tree of Thoughts visualization
- Research mode with parallel sub-query execution
- Memory extraction notifications
- Secret Sauce file browser
